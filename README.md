# GEO Metadata — Data-Quality Findings

Findings from an automated profiling + row-level audit of three public NCBI GEO
supplementary-metadata datasets (GSE131907, GSE181919, GSE157103), organized
**per table**. Each headline finding was re-confirmed by a tool-based verifier
that computes the evidence and judges it against the dataset's documented priors.

Findings are data-quality hypotheses, not assertions about any downstream
publication; impact statements are risks. Severity reflects downstream impact.

📄 **Site:** https://shabtai.github.io/geo-data-quality-findings/

Regenerate the static pages with `python3 build_site.py`.
